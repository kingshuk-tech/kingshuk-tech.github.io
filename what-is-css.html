What is CSS?

CSS stands for Cascading Style Sheets. It is used to design and style websites. While HTML creates the structure of a webpage, CSS makes it look attractive.

Why CSS is Important

Without CSS, websites would look plain and boring. CSS controls colors, fonts, spacing, layout, and responsiveness.

Think of HTML as the skeleton of a body, and CSS as the clothes and appearance.

What Can CSS Do?

  • Change text color
  • Set background images
  • Control spacing and margins
  • Create animations
  • Make websites mobile-friendly

Example of CSS

Here is a simple example:

h1 {
   color: blue;
   font-size: 30px;
}

This makes all h1 headings blue and larger in size.

Types of CSS

  • Inline CSS
  • Internal CSS
  • External CSS

Conclusion

CSS is an essential technology for web development. It works together with HTML and JavaScript to create modern and beautiful websites.
